# Documentação do Compilador

Este diretório contém a documentação técnica do compilador.

Arquivos principais:

- `especificacao.md` — especificação da linguagem, tokens, gramática, e comportamento de cada etapa do compilador.
- `README.md` — este arquivo, índice dos documentos.

Como usar:

- Leia `especificacao.md` para entender a linguagem e as fases do compilador.
- Consulte os módulos em `src/` para ver implementações específicas (lexer, parser, analisador semântico, gerador).

Se quiser que eu gere diagramas (AFD, gramática em BNF) ou um guia de contribuição mais detalhado, me avise.
